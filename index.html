About this project

This MVP focuses on player-level matchplay singles results from the Ryder Cup, Presidents Cup, and WGC Match Play. The data model is ready for additional events and formats.

  • Scope: Singles matches only
  • Events: Ryder Cup, Presidents Cup, WGC Match Play
  • Ranking: Points per match
