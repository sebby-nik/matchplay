About this project

This MVP focuses on player-level matchplay singles results from the Ryder Cup, Presidents Cup, WGC Match Play, and Seve Trophy. The data model is ready for additional events and formats.

  • Scope: Singles matches only
  • Events: Ryder Cup, Presidents Cup, WGC Match Play, Seve Trophy
  • Ranking: Points per match
